API Endpoints

  • GET /health - Health check
  • GET /api/tools - All tools (local + proxied)
  • GET /api/logs - Invocation logs
  • GET /api/proxy/status - Upstream status
  • GET /api/proxy/tools - Proxied tool map
  • POST /api/proxy/reconnect/{name} - Reconnect upstream
  • GET /mcp/sse - MCP SSE endpoint

Proxy Architecture

MCP Hub proxies tools from upstream servers. Each tool is namespaced with a prefix (e.g. github__create_issue, brave__web_search). Configure upstreams in upstreams.yaml.
